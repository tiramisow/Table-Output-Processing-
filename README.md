CSV Data Processing Web App

Overview
This project is a simple web application that reads data from a CSV file, displays it in a table, and performs calculations to generate a second table.

The application demonstrates basic data processing and dynamic rendering using JavaScript.

Features
- Load data from a CSV file
- Display raw data (Table 1)
- Perform calculations:

Alpha = A5 + A20
Beta = A15 / A7
Charlie = A13 * A12
Display computed results (Table 2)

Output:
- Table 1: Displays all values from the CSV file
- Table 2: Displays calculated results for Alpha, Beta, and Charlie
